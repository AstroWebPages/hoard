Introducción

En Hoard queremos que cada vendedor tenga claro cómo funciona nuestro sistema de comisiones y las reglas para garantizar transparencia y cumplimiento para todos.




Costos de Comisión

El servicio de Hoard aplica una comisión base sobre cada venta generada:

  • 10% para ventas dentro de la ciudad capital (local).
  • 15% para ventas a nivel nacional.
  • 25% para ventas internacionales (mundial).

Estas comisiones se calculan sobre el valor total de la venta.




Incumplimiento de Pago

En caso de que un usuario o vendedor se niegue a pagar la comisión acordada por venta efectuada, Hoard se reserva el derecho de reportar la cuenta involucrada en todos los marketplaces, tiendas virtuales y plataformas de venta asociadas para proteger la integridad de nuestra red y de los promotores.

Con Hoard, todos ganamos cuando cumplimos las reglas.
